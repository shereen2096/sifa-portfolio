# Sifa Shereen M — Personal Portfolio

A modern, fully responsive personal portfolio website built with pure HTML5, CSS3, and Vanilla JavaScript. No frameworks, no build step — deployable directly on GitHub Pages.

## Live Site

Deploy to GitHub Pages and share your portfolio with recruiters, internship coordinators, and hiring managers.

## Features

- Dark glassmorphism design
- Typing animation effect
- Scroll reveal animations
- Animated stat counters
- Proficiency bar animations
- Sticky responsive navbar with mobile hamburger menu
- Staggered project card reveals
- Contact form with mailto fallback
- Scroll-to-top button
- Loading screen
- Fully responsive (mobile, tablet, desktop)
- SEO meta tags and Open Graph

## Sections

1. Hero — tagline, typing effect, profile photo, social links, stats
2. About — bio, career objective, contact info, interests
3. Education — timeline with SKCET + Pem School
4. Skills — category cards + proficiency bars
5. Projects — 4 featured project cards
6. Certifications & Achievements
7. Contact — info cards + contact form
8. Footer

## Deployment (GitHub Pages)

1. Push all files to a GitHub repository.
2. Go to **Settings → Pages → Source → Deploy from a branch → main / root**.
3. Your portfolio will be live at `https://<username>.github.io/<repo>/`.

## Customisation

- Update contact details in `index.html`.
- Replace `assets/Sifa_Shereen_Resume.pdf` with your actual resume PDF.
- Adjust colors in `style.css` under `:root` variables.
